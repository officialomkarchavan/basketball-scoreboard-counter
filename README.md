# Basketball Counter

A simple and functional basketball scoreboard app built in 30 minutes.

## Features

- **Score Tracking**: Track points for two teams in real-time
- **Quick Controls**: Fast and intuitive button controls to add or subtract points
- **Reset Function**: Clear scores and start a new game
- **Minimal Design**: Clean, distraction-free interface

## Getting Started

1. Open the app
2. Use the controls to update scores for each team
3. Press reset to start a new game

## Technologies Used

- HTML, CSS and JS fundamentals

## Usage

- **+1, +2, +3 buttons**: Add points to a team's score
- **Reset button**: Clear all scores

## Notes

Built as a quick project to practice rapid development and core functionality.
